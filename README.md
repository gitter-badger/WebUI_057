# WebUI_057 - "SOFTSERVE TESTING"
Our main project
* Install Gulp and plugins
```
    $ npm install
```
* Install AngularJS, Bootstrap and others
```
    $ bower install
```
* Start Gulp watcher
```
    $ gulp
```
* Build project
```
    $ gulp build
```
* Build production version
```
    $ gulp build_production
```
